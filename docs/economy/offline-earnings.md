# Offline Earnings

## Core Rule

Your Bees never sleep.

Offline earnings are required for daily return rate. Players should come back and feel that the Hive kept working while they were gone.

## Base Offline Rules

| Status | Earnings |
|---|---:|
| Online active | 100% |
| Offline base | 20% |
| Offline with Queen upgrades | 25% to 40% |
| Offline cap | 8 hours |
| Overheated miners offline | Reduced or zero output |
| Propolis miners offline | 0% output |

## Offline Claim Screen

When a player returns, show:

- HoneyCoin earned
- Pollen generated
- Blocks attempted
- Blocks found
- Miners overheated
- Propolis events
- Repair warnings
- Royal Jelly earned
- Contracts completed

## Example Claim Message

Your Bees worked while you slept.

HoneyCoin Earned: 42,000  
Pollen Generated: 18,500  
Blocks Found: 2  
Miners Overheated: 1  
Propolis Events: 1  
Royal Jelly Earned: 3

## Offline Formula

Offline HoneyCoin =  
OnlineRate × OfflineEfficiency × HoursAway

Capped at 8 hours by default.

## Queen Upgrade Path

| Queen Upgrade | Offline Efficiency |
|---|---:|
| Sleeping Swarm I | 25% |
| Sleeping Swarm II | 30% |
| Sleeping Swarm III | 35% |
| Royal Night Shift | 40% |

## Design Note

Offline earnings should feel generous enough to create habit, but not so strong that players avoid active play.

Active players should always earn more through:

- Block events
- Heat optimization
- Pool timing
- Contract completion
- Trading
- Hosting
- Tribunal grades
