# MVP Loop

## First Play Session

1. Player enters Garage Hive.
2. Player places Dusty Nectar S1.
3. Nectar Node generates Pollen.
4. Pollen converts to HoneyCoin on first block.
5. Player buys Box Fan.
6. Player buys second Nectar Node.
7. Heat rises.
8. Player upgrades Wall Breaker.
9. Player unlocks Starter Rack.
10. Player feeds the Queen for the first time.

## MVP Features

- Garage Hive
- 6 starter Nectar Nodes
- HoneyCoin
- Pollen
- Heat
- Cooling
- Power
- First block found event
- Queen Core
- Offline mining claim
- Save system
