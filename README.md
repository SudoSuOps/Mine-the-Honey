# Mine the Honey

**Mine the Honey** is the design wiki and knowledge graph for **Swarm & Bee: Proof of Honey**.

Players build a Hive, collect Nectar Nodes, grow their Swarm, mine HoneyCoin, feed the Queen, survive halvings, trade rare miners, and chase Genesis blocks while their Bees work 24/7.

## Core Game Loop

Build the Hive.  
Grow the Swarm.  
Mine HoneyCoin.  
Feed the Queen.  
Find the Block.

## Core Taxonomy

| Term | Meaning |
|---|---|
| Hive | Player-owned datacenter/base, starts as a garage |
| Swarm | Player's fleet of mining hardware |
| Nectar Nodes | Mining hardware units |
| HoneyCoin | Main yield currency |
| Pollen | Work-in-progress block output |
| Propolis | Broken/quarantined/failed miner state |
| Royal Jelly | Prestige and rare upgrade currency |
| Queen | Central AI/command core |
| Tribunal | Grading/audit system |
| Genesis Miner | Earn-only mythic origin miner |
