# Block Found Events

Block Found is the main dopamine moment.

Players generate Pollen over time. When a block is found, Pollen converts into HoneyCoin and bonuses may trigger.

## Core Flow

1. Nectar Nodes generate Pollen.
2. Pollen fills the block progress meter.
3. A block is found.
4. HoneyCoin payout triggers.
5. Bonus roll triggers.
6. Tribunal may grade the block.
7. Player sees a celebration moment.

## Block Types

| Block Type | Meaning |
|---|---|
| Normal Block | Standard payout |
| Lucky Block | Bonus HoneyCoin |
| Royal Block | Chance for Royal Jelly |
| Storm Block | Server-wide event block |
| Tribunal Block | Eligible for grade bonus |
| Genesis Signal | Rare questline trigger |

## Payout Components

HoneyCoin Payout =

- Base block reward
- Pollen contribution
- Pool bonus
- Efficiency bonus
- Tribunal grade bonus
- Event bonus
- Queen bonus

## Block Found UI

The player should see:

- Block Found!
- HoneyCoin earned
- Pollen converted
- Bonus result
- Tribunal grade if eligible
- Any rare drop
- Queen feeding progress

## Server-Wide Block Storm

Block Storm is a timed co-op event.

All players contribute hashrate to a server-wide block.

Rewards split by contribution, but every participant should receive something.

## Design Goal

Block Found should feel like the Hive hit payday.

Players should want to be online when block events fire.
