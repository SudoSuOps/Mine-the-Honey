# Tribunal System

The Tribunal is the proof and grading layer of Mine the Honey.

It evaluates:

- Hive efficiency
- Block quality
- Repair quality
- Miner authenticity
- Hosting reputation
- Pool eligibility

## Grades

| Grade | Meaning |
|---|---|
| D | Dirty, inefficient, risky |
| C | Usable |
| B | Stable |
| A | Efficient |
| S | Elite |
| Genesis | Legendary proof |

## Example Verdict

Tribunal Verdict: A-grade Hive.  
Cooling stable.  
Power waste low.  
Royal Jelly Pool unlocked.

## Design Purpose

The Tribunal makes the game feel like Swarm & Bee instead of a generic mining tycoon.
