# Economy V1

## Purpose

The Mine the Honey economy turns mining into a living Hive system.

Players earn HoneyCoin, generate Pollen, manage Propolis failures, feed the Queen, earn Royal Jelly, and grow their Swarm through smart power, cooling, hosting, trading, and Tribunal grades.

## Core Economy Loops

### 1. Active Mining Loop

Nectar Nodes generate Pollen.

Pollen converts into HoneyCoin when:

- A block is found
- Pool work completes
- A contract completes
- A Block Storm event resolves

### 2. Upgrade Loop

Players spend HoneyCoin on:

- Nectar Nodes
- Rack slots
- Power systems
- Cooling systems
- Repairs
- Hive expansion
- Worker Bees
- Vault upgrades

### 3. Risk Loop

Pushing hash too hard creates heat.

Heat can cause:

- Throttling
- Downtime
- Propolis state
- Repair costs
- Salvage decisions

### 4. Prestige Loop

Players feed the Queen and earn Royal Jelly.

Royal Jelly unlocks:

- Permanent boosts
- Better offline mining
- Miner evolution
- Royal Jelly Pool access
- Tribunal upgrades
- Genesis Miner quest progress

### 5. Social Economy Loop

Players can:

- Trade Nectar Nodes
- Host other players' miners
- Visit Hives
- Compare Tribunal grades
- Join co-op Block Storms
- Compete on leaderboards

## Design Principle

Raw hashpower wins early.

Efficiency wins late.

The best Hive is not the loudest Hive.  
The best Hive is the one that survives the Halving.
