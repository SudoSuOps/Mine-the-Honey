# The Genesis Miner

Long before the Hives, before HoneyCoin, before the Royal Jelly Pools, there was one machine.

It was not the fastest miner.

It was not the loudest miner.

It was the first miner that proved work.

They called it:

**The Genesis Miner**

The Genesis Miner was built by The Architect, the first beekeeper of the Swarm.

The Architect believed that every machine should prove its work, every Hive should earn its place, and every block should carry a receipt.

At first, the world was full of noise:

- broken rigs
- fake blocks
- overheated farms
- empty promises

The Architect created the Tribunal to separate real Honey from waste.

The Tribunal gave three verdicts:

**Honey** — proven yield.  
**Pollen** — work still forming.  
**Propolis** — corrupted, broken, or unsafe.

The Queen was built next.

Not as a ruler, but as a memory core for the Hive.

Players feed the Queen with HoneyCoin and Royal Jelly. In return, she teaches the Swarm how to become smarter, cooler, faster, and more efficient.

Every player begins with a small garage Hive.

One dusty Nectar Node.

One weak breaker.

One noisy fan.

But every Hive carries the same dream:

Find the block.  
Feed the Queen.  
Build the Swarm.

The Genesis Miner cannot be bought.

It must be earned.
