# Offline Mining

Offline mining is required for daily return rate.

## Rule

The Bees never sleep.

## Example Offline Earnings

| Status | Earnings |
|---|---:|
| Online active | 100% |
| Offline base | 20% |
| Offline with Queen upgrade | 25-40% |
| Offline cap | 8 hours |

## Claim Screen

When players return, show:

- HoneyCoin earned
- Blocks attempted
- Blocks found
- Miners overheated
- Propolis events
- Royal Jelly earned
- Contracts completed

## Example Message

"Your Bees worked while you slept."
