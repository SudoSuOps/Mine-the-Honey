# Halving System

The Halving is the long-term prestige clock.

Every season or major cycle, the base reward decreases while difficulty rises. This teaches players that efficiency beats raw greed.

## Example Cycle Table

| Cycle | Base Block Reward | Difficulty Multiplier | Lesson |
|---|---:|---:|---|
| Genesis Cycle | 100% | 1.0x | Learn mining |
| Cycle 1 | 50% | 1.25x | Efficiency matters |
| Cycle 2 | 25% | 1.6x | Weak rigs struggle |
| Cycle 3 | 12.5% | 2.0x | Royal Jelly upgrades matter |
| Cycle 4 | 6.25% | 2.5x | Optimized Hives dominate |

## Payout Formula

HoneyCoin Payout =  
BlockBase × HalvingMultiplier × DifficultyAdjustment  
+ FeeRushBonus  
+ PoolShareBonus  
+ TribunalGradeBonus

## Design Note

Halving should not punish casual players too hard. Later cycles should introduce fee bonuses, pool bonuses, and Queen upgrades to keep the game fun.
