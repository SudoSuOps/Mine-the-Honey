# Block Grade Formula

Block Grade measures the quality of a block found event.

## Inputs

| Input | Meaning |
|---|---|
| Pollen Contribution | How much work the player contributed |
| Efficiency During Block | How cleanly the work was produced |
| Heat Stability | Whether miners stayed stable |
| Pool Contribution | Whether the player helped a pool |
| Event Bonus | Whether the block was found during a special event |
| Queen Bonus | Permanent progression impact |

## Block Grade Outcomes

| Grade | Reward Effect |
|---|---|
| D | Base reward only |
| C | Small HoneyCoin bonus |
| B | HoneyCoin + small Pollen refund |
| A | HoneyCoin + Tribunal Marks |
| S | HoneyCoin + Royal Jelly chance |
| Genesis Signal | Rare questline trigger |

## Design Goal

Players should care about how they found the block, not only that they found it.

A clean block should feel better than a sloppy block.

## Example Block Verdict
