# Nectar Nodes

Nectar Nodes are the mining hardware units of Mine the Honey.

They are inspired by real mining rigs, ASICs, GPUs, hydro miners, and datacenter hardware, but they are fictional and Roblox-safe.

## Core Stats

Each Nectar Node has:

- Hashrate
- Power draw
- Heat output
- Efficiency
- Reliability
- Rarity
- Family
- Skin
- Trade status
- Propolis risk
