# Trading System

Trading is the social economy loop.

Players trade Nectar Nodes, skins, rare recovery variants, event items, and selected materials.

## Why Trading Matters

Visible flex needs an audience.

Rare miners need a market.

Players stay longer when they negotiate, compare, collect, and chase missing items.

## Tradeable Items

| Item Type | Tradeable? | Notes |
|---|---|---|
| Nectar Nodes | Yes | Core trading item |
| Skins | Yes | Cosmetic economy |
| Rare Propolis variants | Yes | Collector market |
| Wax | Limited | May need caps |
| Nectar Shards | Limited | Useful upgrade material |
| Genesis Node | No | Earn-only proof item |
| Starter Nodes | No | Tutorial protection |

## Anti-Scam Requirements

Trading must include:

- Confirm screen
- Item rarity display
- Stat display
- Warning for rare items
- Trade lock timer after major changes
- No hidden item swaps
- Clear acceptance state

## Trade Hub

The Trade Hub is a social room where players can:

- show rare Nectar Nodes
- post trade offers
- inspect miner stats
- compare Hive grades
- find hosting slots
- flex skins and badges

## Design Goal

Trading should extend session length without becoming mandatory.

Players who love collecting should have a reason to stay online.
