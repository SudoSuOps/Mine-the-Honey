# Queen Feeding

The Queen is the central Hive intelligence.

Players feed the Queen with HoneyCoin, Royal Jelly, and high-grade proof to unlock permanent progression.

## Purpose

The Queen gives emotional and mechanical meaning to progression.

Players are not just upgrading a menu.

They are feeding the memory core of the Hive.

## Feeding Inputs

| Input | Meaning |
|---|---|
| HoneyCoin | Basic nourishment and upgrade progress |
| Royal Jelly | Prestige nourishment |
| Tribunal Marks | Proof-quality nourishment |
| Rare Blocks | Milestone nourishment |
| Genesis Signals | Mythic quest nourishment |

## Queen Upgrade Categories

| Category | Effect |
|---|---|
| Hash Memory | Permanent hash boosts |

| Tribunal Memory | Better grade bonuses |
| Propolis Recovery | Better repair outcomes |
| Pool Wisdom | Better pool bonuses |
| Genesis Signal | Unlocks Genesis Miner path |

## Design Rule

Queen upgrades should feel permanent and meaningful.

Royal Jelly should be the currency of memory.

HoneyCoin buys machines.

Royal Jelly evolves the Hive.
