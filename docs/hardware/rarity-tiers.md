# Rarity Tiers

Rarity controls collection value, trade value, animation intensity, and upgrade depth.

## Tiers

| Tier | Color Direction | Trade Value | Notes |
|---|---|---:|---|
| Starter | Plain | None | Tutorial items |
| Common | Simple | Low | Early progression |
| Uncommon | Slight glow | Low-medium | First chase items |
| Rare | Stronger glow | Medium | Meaningful upgrades |
| Epic | Animated parts | High | Midgame chase |
| Legendary | Unique effects | Very high | Flex items |
| Mythic | Custom animation | Extremely high | Rare trading anchors |
| Genesis | Unique proof aura | Earn-only | Cannot be bought |

## Rarity Design

Rarity should affect:

- visual effects
- base stats
- upgrade slots
- skin capacity
- Tribunal bonus chance
- Propolis recovery outcomes
- trade demand

## Rarity Should Not Mean

Rarity should not mean automatic win.

A lower-rarity efficient node can outperform a rare but poorly cooled node.

## Visual Direction

| Tier | Visual Feel |
|---|---|
| Common | simple fan lights |
| Uncommon | colored status LEDs |
| Rare | animated fan glow |
| Epic | pulsing power lines |
| Legendary | liquid/cooling animation |
| Mythic | custom aura and sound |
| Genesis | proof ring, golden comb aura |
