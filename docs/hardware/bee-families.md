# Bee Families

Bee families define miner identity, strengths, and upgrade paths.

| Family | Identity |
|---|---|
| Apis | Balanced all-around miners |
| Bombus | Heavy power, high heat, big payouts |
| Osmia | Efficient, low-power solo miners |
| Megachile | Builder, repair, and crafting specialists |
| Xylocopa | Industrial heavy-frame miners |
| Melipona | Rare stealth-efficiency miners |
| Halictus | Lucky/speculative event miners |
| Andrena | Seasonal/event miners |
| Euglossa | Cosmetic and rare skin collectors |
| Trigona | Swarm bonus/co-op miners |
