# Economy Table

| Economy Item | Earned From | Spent On | Purpose |
|---|---|---|---|
| HoneyCoin | Mining, offline claim, block found, contracts | Miners, racks, power, cooling, repairs | Main money loop |
| Pollen | Active mining progress | Converted into HoneyCoin | Keeps players watching progress |
| Royal Jelly | Prestige, Tribunal-grade blocks, rare events | Queen upgrades, miner evolution, rare pools | Long-term progression |
| Propolis | Overheating, failures, bad overclocking | Repair, quarantine, salvage | Risk/punishment loop |
| Wax | Quests, dismantling old gear | Boards, plugs, skins, crafting | Crafting material |
| Nectar Shards | Events, trading, rare drops | Miner upgrades/chip rolls | Collection chase |
| Tribunal Marks | High-grade efficiency audits | Elite pools, cosmetics, hosting reputation | Skill/status currency |
| Speculative Honey | Risk pools, planned launches, rare events | High-risk upgrades/events | Risk/reward excitement |

## Economy Principle

Raw hashpower should help early. Efficiency, cooling, Tribunal grade, Queen upgrades, and Royal Jelly should win long term.
