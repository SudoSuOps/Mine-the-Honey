# Game Overview

## Working Title

**Swarm & Bee: Proof of Honey**

## One-Line Pitch

A kid-friendly mining datacenter tycoon where Bees run the machines, players build Hives, mine HoneyCoin, feed the Queen, and chase rare Genesis blocks.

## Player Fantasy

Start in a garage with one dusty Nectar Node and build into a massive Hive operation with racks, power systems, cooling, hosting, trading, Royal Jelly upgrades, and Tribunal-graded mining output.

## Main Loop

1. Place Nectar Nodes.
2. Generate Pollen.
3. Convert Pollen into HoneyCoin when blocks are found.
4. Upgrade power and cooling.
5. Expand the Hive.
6. Feed the Queen.
7. Earn Royal Jelly.
8. Survive halvings and block storms.
9. Trade rare miners.
10. Chase the Genesis Miner.

## Session Target

Daily sessions should target 45-90 minutes through:

- Offline mining claims
- Block events
- Heat management
- Miner trading
- Hosting other players' gear
- Queen upgrades
- Royal Jelly progression
- Tribunal grades
