# Hosting Reputation

Hosting Reputation measures whether a player is a trustworthy Hive host.

Players can host other players' Nectar Nodes in their Hive.

The Tribunal tracks whether the host keeps those machines safe, powered, cooled, and productive.

## Reputation Inputs

| Input | Meaning |
|---|---|
| Uptime | Hosted nodes stay online |
| Heat Safety | Hosted nodes avoid overheating |
| Power Stability | Host avoids overloads |
| Fair Rate | Electric rate is not abusive |
| Propolis Events | Hosted nodes avoid Propolis |
| Guest Reviews | Player feedback |
| Contract Completion | Hosting agreement fulfilled |

## Reputation Grades

| Grade | Meaning |
|---|---|
| D | Risky host |
| C | Basic host |
| B | Reliable host |
| A | Trusted host |
| S | Elite datacenter |
| Genesis | Legendary hosting operator |

## Host Listing Display

Each host listing should show:

- Tribunal Hosting Grade
- uptime
- cooling grade
- electric rate
- open rack slots
- accepted node types
- recent Propolis events

## Design Goal

Hosting should create trust.

The best hosts should become known by name.
