# Royal Jelly Progression

## Purpose

Royal Jelly is the long-term progression currency.

HoneyCoin buys the Hive.  
Royal Jelly evolves the Hive.

## How Players Earn Royal Jelly

Players earn Royal Jelly from:

- Feeding the Queen
- Prestige milestones
- Tribunal A/S-grade results
- Rare block discoveries
- Halving events
- Block Storm participation
- Genesis quest progress
- First-time facility unlocks

## How Players Spend Royal Jelly

Players spend Royal Jelly on:

- Queen upgrades
- Permanent hash boosts
- Permanent cooling boosts
- Offline earning upgrades
- Miner evolution
- Royal Jelly Pool access
- Tribunal grade multipliers
- Genesis Miner unlock path

## Queen Upgrade Tree

| Upgrade | Effect |
|---|---|
| Queen Core I | Unlocks offline mining |
| Queen Core II | +5% global hash |
| Queen Core III | +5% cooling efficiency |
| Sleeping Swarm I | Offline mining improves |
| Tribunal Memory I | Better grade bonuses |
| Royal Jelly Pool Key | Unlocks elite pool |
| Propolis Recovery I | Better repair outcomes |
| Genesis Signal I | Starts Genesis Miner questline |

## Prestige Rule

Prestige should reset most HoneyCoin progress but preserve:

- Royal Jelly
- Queen upgrades
- unlocked miner families
- badges
- cosmetics
- Genesis quest progress
- Tribunal history

## Design Principle

Royal Jelly should never feel like normal money.

It should feel like the Queen is remembering the player's work.
