# Propolis Inspection

Propolis Inspection lets the Tribunal evaluate failed or quarantined Nectar Nodes.

This turns broken hardware into gameplay.

## Inspection Choices

When a miner enters Propolis, the player can:

- repair it
- salvage it
- quarantine it
- send it to Tribunal inspection
- use Royal Jelly restoration

## Tribunal Inspection Outcomes

| Outcome | Meaning |
|---|---|
| Failed | Node remains Propolis |
| Repairable | Repair cost reduced |
| Salvage Boost | Better salvage materials |
| Scarred Variant | Rare recovered variant |
| Relic Variant | Very rare collector recovery |
| Genesis Signal | Extremely rare mythic clue |

## Example Recoveries

| Original Node | Possible Recovery |
|---|---|
| Vintage Nectar S9 | Scarred S9 |
| Bombus HeatCore | Charred HeatCore |
| Nectar Hydro | Frostbitten Hydro |
| Royal Hydro Node | Royal Scar Hydro |
| BlackComb Titan | BlackComb Relic |

## Design Goal

Failure should create stories.

A player should remember:

"My S9 almost died, but the Tribunal recovered a Scarred S9."
