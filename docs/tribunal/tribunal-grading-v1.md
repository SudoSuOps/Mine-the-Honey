# Tribunal Grading V1

The Tribunal is the proof and grading layer of Mine the Honey.

It turns raw mining output into judged, trusted, high-quality Hive performance.

## What the Tribunal Grades

| Grade Target | Meaning |
|---|---|
| Hive Grade | Overall efficiency and stability of the player's Hive |
| Block Grade | Quality of a found block event |
| Miner Grade | Condition and performance of a Nectar Node |
| Hosting Grade | Trust score for player hosting |
| Repair Grade | Quality of Propolis recovery |
| Pool Grade | Eligibility for higher-yield pools |

## Grade Scale

| Grade | Meaning |
|---|---|
| D | Dirty, unstable, wasteful |
| C | Usable but inefficient |
| B | Stable |
| A | Efficient and reliable |
| S | Elite |
| Genesis | Legendary proof status |

## Core Inputs

Tribunal grades should consider:

- hash output
- power efficiency
- heat stability
- uptime
- Propolis events
- successful repairs
- block contribution
- hosting reliability
- Queen upgrade level
- event participation

## Design Principle

The Tribunal should reward players who run clean Hives.

Raw hash can win a short race.

Tribunal grade wins the long game.
